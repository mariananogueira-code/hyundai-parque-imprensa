# Hyundai — Parque de Imprensa

Plataforma de gestão do parque de imprensa da Hyundai Portugal.

## Setup

1. Fazer deploy dos ficheiros para GitHub Pages
2. Aceder via `https://{username}.github.io/{repo-name}/`

## Acesso

- **Admin**: `#hpi-admin-2026` (acesso total)
- **Editor**: `#hpi-editor-2026` (editar marcações)
- **Viewer**: `#hpi-viewer-2026` (apenas consulta)

## Stack

- React 18 (CDN)
- Supabase (PostgreSQL + REST API)
- GitHub Pages (hosting)
